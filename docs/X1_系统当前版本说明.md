# X1 检测报告生成系统 — 当前版本说明

**版本号**: X4.8  
**更新时间**: 2026-05-16  
**运行环境**: macOS (Mac mini) / Python 3.9 + Flask  
**服务端口**: 8082  

---

## 系统定位

X1 是内蒙古普迪检测技术服务有限公司的检测报告正式生产系统，覆盖从现场数据录入、判定计算、报告生成到飞书归档的完整业务链。

## 当前能力范围

### 检测对象覆盖（14 类）

| 领域 | 对象 |
|------|------|
| 制药工业 | GMP车间、兽药GMP车间、传递窗、层流罩 |
| 医院洁净部 | 洁净手术部（百/千/万/十万级）、洁净功能用房、负压病房 |
| 生物安全 | 生物安全实验室、动物房、生物安全柜、洁净工作台、IVC笼具 |
| 食品加工 | 食品洁净车间 |
| 精密制造 | 电子洁净车间 |

### 核心功能

1. **现场录入** — 移动端/桌面端参数录入，支持草稿保存与恢复
2. **智能判定** — 基于国标的自动合格/不合格判定引擎
3. **报告导出** — docx 检测报告 + xlsx 原始记录，模板绑定导出
4. **飞书归档** — 自动上传至飞书云空间，按月份目录归档
5. **客户管理** — 客户注册、项目管理、报告查询
6. **模板治理** — 模板注册、验证、启停、语义映射
7. **权限管理** — 角色权限体系，操作日志审计

### 已知边界

- PDF 转换依赖 macOS Pages.app（仅 desktop 模式可用）
- 飞书上传需要有效的 app_id/app_secret 配置
- 当前为单机部署，不支持集群

## 技术栈

- 后端：Python 3.9 / Flask / SQLite
- 前端：原生 JS + esbuild 构建
- 模板引擎：python-docx / openpyxl
- 部署：macOS LaunchAgent 守护进程

# X1 常见问题排障手册

**更新时间**: 2026-05-16

---

## 1. 服务无法启动

**现象**: 执行 `start_x1_daemon.sh` 后服务未运行

**排查步骤**:
1. 检查端口占用：`lsof -nP -iTCP:8082 -sTCP:LISTEN`
2. 检查 PID 文件：`cat x1.pid`，确认进程是否存在
3. 查看应用日志：`tail -50 logs_x1/app_$(date +%Y%m%d).log`
4. 手动启动看报错：`python3 app_x1.py`

**常见原因**:
- 端口被其他进程占用 → 先停止占用进程
- Python 依赖缺失 → `python3 -m pip install -r requirements.txt`
- 配置文件损坏 → 检查 `x1_config.json` 格式

---

## 2. 端口被占用

**快速处理**:
```bash
# 查看谁占了 8082
lsof -nP -iTCP:8082 -sTCP:LISTEN
# 杀掉占用进程
kill <PID>
# 重新启动
bash start_x1_daemon.sh
```

---

## 3. 模板缺失 / 导出报告失败

**现象**: 导出时提示"模板不存在"或生成空报告

**排查步骤**:
1. 确认模板根目录存在：`ls ~/公司资料/检测部/检测报告模板/`
2. 检查 `x1_config.json` 中 `template_base` 路径是否正确
3. 后台 → 模板池，查看各类型的"有效"数量
4. 如果路径含 `~`，确认代码中有 `expanduser()` 处理

---

## 4. 飞书上传失败

**排查步骤**:
1. 确认飞书开关：`x1_config.json` → `feishu.enabled: true`
2. 确认凭证有效：检查 `feishu_config.json` 中 app_id/app_secret
3. 测试 token 获取：
   ```python
   from feishu_utils import get_feishu_token
   print(get_feishu_token())  # 应返回非空字符串
   ```
4. 测试文件夹访问：
   ```python
   from feishu_utils import resolve_feishu_upload_folder
   print(resolve_feishu_upload_folder('reports'))  # 应返回 folder token
   ```
5. 检查飞书应用权限是否包含云空间读写

**常见原因**:
- Token 过期 → 自动刷新，一般无需手动处理
- 文件夹 token 失效 → 更新 `feishu_config.json` 中的 folders
- 网络不通 → 检查是否能访问 `open.feishu.cn`

---

## 5. 数据库锁定 / 写入失败

**现象**: 操作报错 "database is locked"

**处理**:
```bash
# 检查是否有多个进程访问数据库
fuser x1_data.db
# 如果有僵尸进程，杀掉后重启
bash restart_x1_daemon.sh
```

---

## 6. 页面加载异常 / JS 报错

**排查步骤**:
1. 强制刷新浏览器缓存（Cmd+Shift+R）
2. 检查 `static/dist/record.bundle.js` 是否存在
3. 如需重新构建前端：`npx esbuild static/src/record/index.js --bundle --outfile=static/dist/record.bundle.js --format=iife --target=es2018`

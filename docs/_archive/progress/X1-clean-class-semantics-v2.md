# X1 洁净等级语义层（第二版）

## 本轮目标
继续把“洁净等级不是普通字符串”这件事落实到 X 结构中，不只停留在 `operating_room`，开始扩到 `clean_function_room` 和 `bsl`。

---

## 本轮新增

### 1. clean_function_room 进入等级语义层
- 领域：`hospital`
- 标准：`GB 50333-2013`
- 关键点：
  - 洁净功能用房不能只看等级字符串
  - 需要结合 `clean_function_subroom` 一起理解
- 当前语义说明：
  - “医院洁净功能用房等级表达需要结合子房间语义理解，不能脱离子房间独立看待。”

---

### 2. bsl 进入等级语义层
- 领域：`biosafety`
- 标准：`GB 50346-2011 + GB 19489-2008`
- 关键点：
  - `BSL等级` 不是 `洁净等级`
  - 生物安全等级与 ISO 洁净等级不能混成一个概念
- 当前语义说明：
  - “生物安全对象中，BSL等级与洁净等级不是同一概念，洁净等级通常还需与ISO表达联合理解。”

---

## 当前意义
这一步的意义不是已经把这两个对象做完，而是开始从结构上避免以后再把：
- 子房间语义
- BSL等级
- 洁净等级
- ISO表达

混成一个扁平字符串处理。

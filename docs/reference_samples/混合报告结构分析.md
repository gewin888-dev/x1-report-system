### 关键发现：表格结构由级别决定

- **Ⅰ级（百级）**：截面平均风速 + 风速不均匀度 + 开门洁净度 + 手术区/周边区（或局部区/周边区）分区洁净度 + 照度均匀度
- **Ⅱ级（千级）**：截面平均风速 + 手术区/周边区分区洁净度
- **Ⅲ级（万级）**：换气次数 + 手术区/周边区分区洁净度
- **Ⅳ级（十万级）**：换气次数 + 整体洁净度（无分区）

## 三、数据表通用结构（每房间）

```
行0: [房间名称] [名称值] [检测日期] [...] [日期值] [洁净度设计级别] [...] [级别值]
行1: [房间参数] [面积S=xx  体积V=xx] (合并单元格)
行2: [项目] [单位] [标准（设计）要求] [...] [检测结果] [...] [单项结论]
行3~N: 各检测项目数据行
```

### 检测项目完整清单（按级别）

**Ⅰ级必测项（手术室）：**
1. 截面平均风速 (m/s) — 0.20～0.25
2. 风速不均匀度 (/) — ≤0.24
3. 静压差 (Pa) — 对洁净走廊 5～20 / 对污物走廊 5～20
4. 开门后门内0.6m处洁净度 — ≥0.5μm≤35200 / ≥5μm≤293 (ISO-6)
5. 送风高效过滤器检漏 (%) — ≤0.01
6. 严密性 — 除门缝外无可见泄漏
7. 洁净度（手术区）— ≥0.5μm≤3500 / ≥5μm≤0 (ISO-5)
8. 洁净度（周边区）— ≥0.5μm≤35200 / ≥5μm≤293 (ISO-6)
9. 温度 (℃) — 21～25
10. 相对湿度 (%) — 30～60
11. 照度 (lx) — 最低值≥350 + 均匀度≥0.7
12. 噪声 dB(A) — ≤51
13. 细菌浓度（手术区）— ≤0.2
14. 细菌浓度（周边区）— ≤0.4

**Ⅳ级必测项（手术室）：**
1. 换气次数 (次/h) — ≥12
2. 静压差 (Pa) — 对洁净走廊 / 对污物走廊
3. 送风高效过滤器检漏 (%) — ≤0.01
4. 严密性
5. 洁净度 — 整体（无分区）
6. 温度 / 相对湿度
7. 照度（最低值≥350 + 均匀度≥0.7）
8. 噪声 — ≤49
9. 细菌浓度 — ≤6

## 四、洁净度数据格式

### 洁净度行结构（合并单元格复杂）

```
标题行: [洁净度级别（悬浮粒子数)] [粒/m³] [悬浮粒子最大允许数/m³] [...] [洁净度级别] [悬浮粒子数/m³] [...] [洁净度级别] [合格]
数据行: [洁净度级别（悬浮粒子数)] [粒/m³] [区域名] [≥0.5μm:≤3500] [...] [ISO-5] [各采样点平均值中的最大值/95%置信度] [数值] [ISO-5] [合格]
```

### 洁净度数据呈现方式
- 每个粒径（≥0.5μm / ≥5μm）各两行：最大值 + 95%置信度
- 分区时（手术区/周边区/局部区）每区各有独立的粒径行
- 标准要求列同时显示限值和 ISO 等级

## 六、关键学习点

1. **一份报告 = 共享前页 + N个房间数据表**，房间数量不固定
2. **每个房间独立一页**，有自己的"检测报告"标题和报告编号
3. **表格结构由洁净度级别决定**，不同级别检测项目不同、行数不同
4. **报告编号全文重复**：封面1次 + 编制页1次 + 每个房间页各1次
5. **辅房和手术室用不同模板**：辅房Ⅰ级用"局部区/周边区"，手术室Ⅰ级用"手术区/周边区"
6. **洁净度数据最复杂**：合并单元格 + 分区 + 双粒径 + 双统计量（最大值/95%置信度）
7. **项目概述是自动生成的**：格式为"本次委托检测项目为{项目名称}，项目地点位于{地址}，本次检测区域为{区域}。"
8. **气象条件格式固定**：`温度：{T}℃      湿度：{RH}%RH    大气压力：{P}hPa`
